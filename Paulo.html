Linguagens da programação

    Linguagem de estrutura:

    O html é o corpo do site, literalmente a estrutura dele. Se o site fosse uma loja, o html seria o formato da loja.


    Linguaguem de estilização:

    CSS é a linguagem que molda o visual do site, é a etapa que faz seu site ser conhecido. Considerando o outro exemplo, o CSS é responsável pela decoração da loja, as cores, imagens, etc…


    Linguagem de scripit:

    JavaScript é a parte lógica por traz de tudo, aquele que faz as funções funcionarem como deviam. Supondo seu papel em uma loja, provalvemente seria os trabalhadores da loja e a economia dela.
